What is Apache Ray

Ray, proposed in 2017, is a general-purpose cluster-computing framework targeted towards ML/RL data processing workloads. It implements a unified interface of both tasks and actors.

  • Tasks: stateless computation that tried on any server. Returns a `future` which allows user to express parallelism and dependencies.
  • Actor: stateful computation (i.e., class) that exposes methods that can be invoked remotely. Exposes shared mutable state to clients.
